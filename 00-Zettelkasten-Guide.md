## 🎯 Zettelkastenとは
原子的な知識の単位（Zettel）を相互にリンクさせることで、創発的な思考を促進する知識管理システム

## 📁 フォルダ構成
```
00-Inbox/           # 📥 一時的なメモ・アイデア置き場
01-Permanent-Notes/ # 🧠 永続的な知識ノート
02-Literature-Notes/# 📚 文献・記事から学んだこと
03-Reference/       # 📖 参考資料・事実情報
04-Projects/        # 🚀 進行中のプロジェクト
05-Archive/         # 📦 完了・不要になったもの
Templates/          # 📝 各種テンプレート
```

## 🏷️ タグシステム
### 状態タグ
- `#status/draft` - 下書き
- `#status/review` - レビュー中
- `#status/complete` - 完成

### 分野タグ
- `#tech/security` - セキュリティ
- `#tech/programming` - プログラミング
- `#career/strategy` - キャリア戦略
- `#learning/toefl` - TOEFL学習

### 種類タグ
- `#type/concept` - 概念・理論
- `#type/process` - プロセス・手順
- `#type/insight` - 洞察・気づき

## 🔗 リンク戦略
1. **明示的リンク**: `[[ファイル名]]`
2. **文脈リンク**: 関連する概念への自然な流れ
3. **逆リンク**: バックリンクを活用した発見

## 📝 ワークフロー
1. **Inbox**: アイデア・メモを即座にキャプチャ
2. **Processing**: 定期的にInboxを整理
3. **Elaboration**: 永続的ノートに発展
4. **Connection**: 既存ノートとのリンク作成
5. **Evolution**: 継続的な更新・発展

## 🎨 テンプレート使用法
- `Ctrl+T`: テンプレート挿入
- 新規ノート作成時に適切なテンプレートを選択

## 🔍 定期レビュー
- **週次**: Inboxの整理
- **月次**: リンク構造の見直し
- **四半期**: アーカイブ作業
